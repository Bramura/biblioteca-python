# 📚 Sistema de Biblioteca em Python

Projeto de gerenciamento de biblioteca desenvolvido em Python com SQLite, utilizando arquitetura modular e conceitos de CRUD.

---

## 🚀 Funcionalidades

- Cadastro de livros
- Listagem de livros
- Busca de livro por ID
- Remoção de livros
- Atualização de livro
- Empréstimo de livros
- Devolução de livros
- Controle de disponibilidade

---

## 🧠 Conceitos aplicados

- Programação Orientada a Objetos (POO)
- CRUD completo (Create, Read, Update, Delete)
- Manipulação de banco de dados SQLite
- Arquitetura em camadas (models, services, utils)
- Separação de responsabilidades
- Validação de regras de negócio

---

## 🛠️ Tecnologias

- Python 3
- SQLite3 (banco local)
- Programação procedural + POO

---

## 📂 Estrutura do projeto


- models/ → Entidades do sistema (Livro)
- services/ → Regras de negócio e acesso ao banco
- utils/ → Interface de menu
- database.py → Conexão e criação do banco
- main.py → Ponto de entrada do sistema
